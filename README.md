# heart-disease-ml
A machine learning project to predict heart disease using multiple algorithms including Logistic Regression, KNN, Decision Tree, and Random Forest, with performance evaluation using accuracy, precision, recall, F1-score, confusion matrix, and cross-validation.
# ❤️ Heart Disease Prediction using Machine Learning

## 📌 Overview
This project aims to predict the presence of heart disease using machine learning techniques. Multiple classification algorithms are applied and compared to determine the most effective model.

## 📊 Dataset
The dataset is taken from Kaggle (UCI Heart Disease dataset) and includes medical attributes such as age, cholesterol, blood pressure, and more.

## ⚙️ Algorithms Used
- Logistic Regression
- K-Nearest Neighbors (KNN)
- Decision Tree
- Random Forest

## 📈 Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1 Score
- Confusion Matrix
- Cross Validation

## 🏆 Results
Among all models, Random Forest achieved the best performance with higher accuracy and stable cross-validation scores.

## ✅ Conclusion
Random Forest is the most suitable algorithm for heart disease prediction in this project.
